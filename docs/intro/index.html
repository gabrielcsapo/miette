Getting started

Let's discover Miette in less than 5 minutes.

Getting Started#

Get started by installing Miette.

npm install miette --save

Instrument your functions#

Using miette you can make custom error objects that allow you to

@miette("foo::bar::baz", FooBarBaz.toString())class ShouldBeFalseError extends Error {  diagnostic = {    help: "Please consult the guides at http://github.com/foo/bar#guides",  };
  snippets = [    {      context: "if (true)",      highlight: "This will always be called",    },  ];}
function FooBarBaz() {  // eslint-disable-next-line no-constant-condition  if (true) {    try {      throw new ShouldBeFalseError("Should make things dynamic");    } catch (error) {      if (error instanceof Error) {        console.log(error.stack);        expect(error.stack).toMatchSnapshot();      }    }  }}
FooBarBaz();
